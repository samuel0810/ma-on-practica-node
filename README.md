# ma-on-practica-node

En esta actividad, el estudiante deberá seleccionar una plantilla web gratuita del sitio HTMLCodex (https://htmlcodex.com/ Links to an external site.) y adaptarla para que funcione dentro de un servidor web desarrollado con Node.js y Express. Posteriormente, deberá desplegar la aplicación en la plataforma de hosting en la nube Render.com.

El propósito de esta práctica es que el estudiante comprenda el proceso completo de:

    Integración de una plantilla HTML estática dentro de un entorno backend.

    Configuración de un servidor web utilizando Express.

    Organización adecuada de archivos estáticos (HTML, CSS, JS, imágenes).

    Uso de control de versiones con Git y publicación en GitHub.

    Despliegue de aplicaciones Node.js en un servidor en la nube.

Al finalizar la actividad, el estudiante habrá implementado correctamente una aplicación web funcional accesible desde internet, comprendiendo el flujo que va desde el desarrollo local hasta el entorno de producción.
